# Property Investment Calculator - Final Revision

## Run locally

```bash
cd ~/Downloads/property-calculator-final-rev
npm install
npm run dev
```

Open: http://localhost:5173

## Included

- Nima Rahimi logo in `public/logo.png`
- Address sections for both scenarios
- Armstrong Creek what-if-hold comparison from the holding start date
- New property auto loan calculation: purchase price × target LVR
- Separate Bankwest cash-out loan
- Automatic stamp duty estimate by state + purchase price
- LMI estimate with override
- Land tax estimate with override
- Negative gearing cutoff for established properties purchased after 12/05/2026
- Brand-new exception only if year built equals purchase year
- 10, 12 and 15-year comparison
- Formula audit tab

## Important

This is a planning model. Stamp duty, land tax, LMI and tax calculations should be independently verified before making a purchase decision.
